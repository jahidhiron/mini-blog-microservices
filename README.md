## Installation

Install Docker
Install Kubernetes
Install Skaffold
Install MongoDb NoSql Database

```bash
git clone https://github.com/jahidhiron/mini-blog-microservices.git
cd mini-blog-microservices
```

## Usage

```bash
skaffold dev
```
